# BlogHub
用毕生所学，建造一个博客网站
> mysql + mybatis + java8 + springboot + elk + velocity

> restFul风格

> 做全文搜索

###一期目标
> 快速实现Blog基本功能，能够登录、写博文，游客可以访问

##blog-core
> 引入的框架都在这里，通用工具类都在这里

##blog-web
> 用于博客WEBAPP启动项目，依赖blog-core，实现业务领域逻辑，简单配置

##blog-velocity
> 这个项目帮助我们自动生成代码

##未来微服务
> 未来的微服务，项目命名为blog-web-[xx], xx一般是业务关键字
